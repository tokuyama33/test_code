# 仮テスト仕様書

## 概要
課題用の仮テストケース。
対象として、日本郵便のサイトを利用する。
良く利用される機能について、2項目のテストケースを作成する。

## テスト範囲
・トップページからの郵便番号検索
・トップページからの荷物問合せ

## 機能仕様
TBU
